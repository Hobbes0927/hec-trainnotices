# hec-trainnotices

### **Description**

Create player notifications when train updates are received from the HEC Trains Discord bot


### **Features**

- Checks REDM database for new train board entries from Discord bot
- Displays new train updates on player's screen


### **Dependencies**

[vorp_core](https://github.com/VORPCORE/vorp-core-lua)


### **Installation**

- Download the latest master branch from https://github.com/Hobbes0927/hec-trainnotices
- Add hec-trainnotices folder to your resources folder
- Add ensure hec-trainnotices to your resources.cfg
- Restart server
